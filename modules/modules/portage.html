Module portage

present [[atom, version, deep, newuse, nodeps, noreplace, oneshot, onlydeps, sync, update, unmask]]
Install package atom. See emerge(1).

Parameters:

  • atom package atom to install. Can be "category/package" or "category/package-version" [REQUIRED] [ALIAS: package]
  • version package version
  • deep evaluate entire dependency tree [CHOICES: "yes","no"]
  • newuse reinstall packages that had a change in its USE flags [CHOICES: "yes","no"]
  • nodeps do not merge dependencies [CHOICES: "yes","no"]
  • noreplace skip already installed packages [CHOICES: "yes","no"]
  • oneshot do not update the world file [CHOICES: "yes","no"] [DEFAULT: "yes"]
  • onlydeps only merge dependencies [CHOICES: "yes","no"]
  • sync perform an emerge --sync before installing package(s) [CHOICES: "yes","no"]
  • update update package to the best version [CHOICES: "yes","no"]
  • unmask enable auto-unmask and auto-unmask-write options [CHOICES: "yes","no"]

Aliases:

  • installed
  • install

Usage:

     portage.present [[
   package "dev-util/strace"
   version "4.8"
 ]]
 portage.present [[
   atom "=dev-util/strace-4.8"
 ]]
 portage.present [[
   package "dev-util/strace"
 ]]
absent [[atom, depclean]]
Remove package atom.

Parameters:

  • atom package atom to unmerge [REQUIRED] [ALIAS: package]
  • depclean Remove packages not associated with explicitly installed packages [CHOICES: "yes","no"] [DEFAULT: "no"]

Aliases:

    remove

Usage:

     portage.absent [[
   atom "dev-util/atom"
 ]]
