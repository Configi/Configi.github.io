Functions

render(src, table, lua, mode, diff)
Render a textfile.

Subject:

    output file

Note:

    Requires the diffutils package for the diff parameter to work

Parameters:

  • src source template [REQUIRED] [ALIAS: template]
  • table [REQUIRED] [ALIAS: view]
  • lua [ALIAS: data]
  • mode mode bits for output file [DEFAULT: "0600"]
  • diff show diff [CHOICES: "yes","no"]

Usage:

    textfile.render("/etc/something/config")
    template: "/etc/something/config.template"
    view: "view_model"
    data: "/etc/something/config.lua"
insert_line(line, inserts, pattern, plain, before, after)
Insert lines into an existing file.

Subject:

    path of text file to modify

Parameters:

  • line text to insert [REQUIRED] [ALIAS: text]
  • inserts a line (string) if found, skips the operation
  • pattern line is added before or after this pattern [ALIAS: match]
  • plain turn on or off pattern matching facilities [CHOICES: "yes","no"] [DEFAULT: "yes"]
  • before [CHOICES: "yes","no"] [DEFAULT: "no"]
  • after [CHOICES: "yes","no"] [DEFAULT: "yes"]

Usage:

    textfile.insert_line("/etc/sysctl.conf")
    pattern: "# http://cr.yp.to/syncookies.html"
    text: "net.ipv4.tcp_syncookies = 1"
    after: "true"
    plain: "true"
remove_line(pattern, plain)
Remove lines from an existing file.

Subject:

    path of text file to modify

Parameters:

  • pattern text pattern to remove [REQUIRED] [ALIAS: match]
  • plain turn on or off pattern matching facilities [CHOICES: "yes","no"] [DEFAULT: "yes"]

Usage:

    textfile.remove_line("/etc/sysctl.conf")
    match: "net.ipv4.ip_forward = 1"
    plain: "true"
