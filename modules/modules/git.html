Module git

Ensure that a Git repository is cloned into a specified path.

Usage:

     git.repo [[
   repo "https://github.com/torvalds/linux.git"
   path "/home/user/work"
 ]]

pull [[repository, path]]
Run git pull for a repository. This always attempts to run the command. Useful as a handler.

Parameters:

  • repository The URL of the repository. [ALIAS: url,repo] [REQUIRED]
  • path absolute path where to run the pull command [REQUIRED]

Usage:

     git.pull [[
   repo "https://github.com/torvalds/linux.git"
   path "/home/user/work"
 ]]
