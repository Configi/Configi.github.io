unpack [[src, dest, creates]]
Unpack a tar, zip or rar archive.

Parameters:

  • src path to archive [REQUIRED] [ALIASES: archive, tar, zip, rar]
  • dest path where the archive should unpacked [REQUIRED] [ALIAS: directory]
  • creates path to a file, if already existing will skip the unpacking step

Aliases:

  • untar
  • unzip
  • unrar

Note:

    detects archive type using the 3 letter filename extension e.g. tar, zip, rar

Usage:

     unarchive.unpack [[
   src "/tmp/file.tar"
   dest "/tmp/test"
   creates "/tmp/test/file.1"
 ]]
