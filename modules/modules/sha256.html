Module sha256

Check that a given hash matches the actual SHA256 hash of a file.


Useful for alerting on changed hashes

Usage:

     sha256.verify [[
   path "/etc/passwd"
   hash "09e40b7b232c4abb427f1344e636e44ebf5684f70fb6cd67507e88955064255d"
 ]]
