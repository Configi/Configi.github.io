command [[string, cwd, env, creates, removes]]
Run a command via execve(3). shell.command [[ cwd "/tmp" env "test=this whatever=youwant" command "touch test" creates "test" ]]

Parameters:

  • string command to execute [REQUIRED] [ALIAS: command]
  • cwd current working directory
  • env space separated environment variables
  • creates a filename, if found will not run the command
  • removes a filename, if not found will not run the command
system [[string, creates, removes]]
Run a script or command via os.execute.
STDIN and STDERR are closed and STDOUT is piped to /dev/null

Parameters:

  • string script or command to execute [REQUIRED] [ALIAS: script,command]
  • creates a filename, if found will not run the script
  • removes a filename, if not found will not run the script

Aliases:

    script

Usage:

     shell.system [[
   script "/root/test.sh"
 ]]
popen [[string, cwd, creates, removes, expects]]
Run a command via io.popen.

Parameters:

  • string command to execute [REQUIRED] [ALIAS: command]
  • cwd current working directory
  • creates a filename, if found will not run the command
  • removes a filename, if not found will not run the command
  • expects instead of the exit code, use a string match as a test for success

Usage:

     shell.popen [[
   cwd "/tmp"
   command "ls -la"
   expects ".X11-unix"
 ]]
popen3 [[string, cwd, env, creates, removes, stdin, stdout, stderr, error]]
Run a command via Px.exec which can expect strings from STDIN, STDOUT or STDERR

Parameters:

  • string command to execute [REQUIRED] [ALIAS: command]
  • cwd current working directory
  • env space separated string of environment variables
  • creates a filename, if found will not run the command
  • removes a filename, if not found will not run the command
  • stdin pipe a string from STDIN
  • stdout test for a string from STDOUT
  • stderr test for a string from STDERR
  • error ignore errors when set to "ignore" [CHOICES: "yes","no"]

Usage:

     shell.popen3 [[
   cwd "/tmp"
   command "ls"
   stdout ".X11-unix"
 ]]
