attributes [[path, mode, owner, group]]
Set path attributes such as the mode, owner or group.

Parameters:

  • path path to modify [REQUIRED]
  • mode set the file mode bits
  • owner set the uid/owner [ALIAS: uid]
  • group set the gid/group [ALIAS: gid]

Usage:

     file.attributes [[
   path "/etc/shadow"
   mode "0600"
   owner "root"
   group "root"
 ]]
link [[src, path, force]]
Create a symlink.

Parameters:

  • src path where the symlink points to [REQUIRED]
  • path the symlink [REQUIRED] [ALIAS: link]
  • force remove existing symlink [CHOICES: "yes","no"]

Usage:

     file.link [[
   src "/"
   path "/home/ed/root"
 ]]

directory [[path, mode, owner, group, force, backup]]
Create a directory.

Parameters:

  • path path of the directory [REQUIRED]
  • mode set the file mode bits
  • owner set the uid/owner [ALIAS: uid]
  • group set the gid/group [ALIAS: gid]
  • force remove existing path before creating directory [CHOICES: "yes","no"] [DEFAULT: "no"]
  • backup rename existing path and prepend '.configi' to the name [CHOICES: "yes","no"] [DEFAULT: "no"]

Usage:

     file.directory [[
   path "/usr/portage"
 ]]

copy [[path, src, recurse, force, backup]]
Copy a path.

Parameters:

  • path destination path [REQUIRED] [ALIAS: dest,target]
  • src source path to copy [REQUIRED]
  • recurse recursively copy source [CHOICES: "yes","no"] [DEFAULT: "no"]
  • force remove existing destination before copying [CHOICES: "yes","no"] [DEFAULT: "no"]
  • backup rename existing path and prepend '.configi' to the name [CHOICES: "yes","no"] [DEFAULT: "no"]

Usage:

     file.copy [[
   src "/home/ed"
   dest "/mnt/backups"
 ]]
